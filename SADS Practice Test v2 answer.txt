A6. Logical Volume Manager
-create LVM volume group named vg1 from the 2 Linux LVM partition size 8MB
-create logical volume named lv1 with the size of 30 physical extends
-create an ext4 file system on lv1
-Mount the new file system at /mnt/lv1 persistently
#pvcreate /dev/sdb5 /dev/sdd6
#vgcreate vg1 -s 8M /dev/sdb5 /dev/sdb6
#lvcreate -n lv1 -l 30 /dev/vg1
#lvdisplay (take note of LV PATH)
#mkfs -t ext4 /dev/vg1/lv1

#mkdir /mnt/lv1
#vi /etc/fstab

Add in /etc/fstab
==================
/dev/vg1/lv1 /mnt/lv1 ext4 defaults 0 0
:wq

#systemctl daemon-reload
#mount -a
#lsblk (show what has been done)

A8. 
-Schedule a system cron job that runs every wed at 11:45pm using "jenny" account. The cronjob creates a bzip2 tar archive (filename ~/DDMMYY-home.tar.bz2).
-schedule a system cron job that appends date and time to /tmp/share/cron-output.txt every 1st day of the month runnung as root account.

#vi /etc/crontab

add in /etc/crontab
===================
45 23 * * wed jenny tar -cjf /home/jenny/$(date \+\%d\%m\%y)-home.tar.bz2 /home/jenny
*/1 * * * *  jenny tar -cjf /home/jenny/$(date \+\%d\%m\%y)-home.tar.bz2 /home/jenny (test every min)
* 0 1 * * root echo $(date) >> /tmp/share/cron-output.txt
:w:q
